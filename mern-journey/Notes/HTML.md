What is HTML?
HTML (hypertext markup language) is a standard language used to create structure of a web page.

What is a tag?
Tags are special keywords enclosed in angle brackets (<>) that tell the browser how to display content.

What is an element?
An html element consist of: opening tag, content, closing tag.

What are Attributes?
attributes provide extra information about html element. Ex- href,src,alt.

What is <!DOCTYPE html>?
the <!DOCTYPE html> tells the browser that the document is written using HTML5, which is the latest verson of html.

What does the <head> section contain?
The <head> contains information about the webpage rather than the webpage's visible content.

What does the <body> section contain?
The <body> section contains everything that is visible on the webpage, Whatever you see on a webpage is generally inside the <body>.

## HTML Headings

What is a heading?
it is a title or subtitle that organizes webpages content.

Why do we use headings?
To organize content, improve readability, hlep search engines understand the page, supports accessibility.

Difference between <h1> and <h6>.
<h1> is the highest-level and most important heading; <h6> is the lowest-level and least important heading.

How many <h1> tags should a page usually have?
usually one main<h1> per page, with lower-level headings used for sections and subsections.

Why shouldn't headings be used just to make text look bigger?
headings describe the structure and meaning of the content, if you only want to change appearance then use CSS.

## HTML Links

What is the <a> tag?
the <a> tag, also called the Anchor tag, is  used to create hyperlinks in html.

What is the href attribute?
href stands for hypertext refrence. it tells the browser where the link should go when click.

What does target="_blank" do?
target="_blank" opens the linked page in new tab.

What is the difference between an absolute link and a relative link?
an absolute link contains the complete web address(URL). a relative link points to another file inside your project.

What is a mailto: link?
a mailto: links opens the users default email application with the recipients email address already filled in.

What is a tel: link?
a tel: links allows users to call a phone number directly from supported devices.

## HTML Images

What does the src attribute do?

Why is the alt attribute important?

Why should images be stored in a separate folder?
using a images folder keeps the project clean. every developer stores images separately.
